About

  • The main aim of this website is to predict the online transaction frauds.
  • The credit card transaction details are entered in the fields provided and a click on predict button will give the result .
  • The output is displayed whether the entered transaction is a fraud or not.
  • The step,type,amount,oldbalanceOrig,newbalanceOrig,oldbalanceDest, newbalanceDest are the basis for the prediction.
  • The ML model random forest classifier is used for prediction with fraud transaction CSV file as dataset.
